# Video minh họa thủ ấn

Bỏ clip tay thật vào đây, đặt đúng tên file thì video sẽ tự hiện trong lúc tập
trên trang Relaxation (pages/hoi-tho.html). Không cần sửa code.

Tên file bắt buộc (mp4):

- varuna.mp4   → Ấn Varuna
- apana.mp4    → Ấn Apana
- vata.mp4     → Ấn Vata Naashak
- vayu.mp4     → Ấn Vayu
- gyan.mp4     → Ấn Gyan

Gợi ý khi quay:
- Mỗi clip 10-20 giây, xoay chậm bàn tay để thấy rõ các góc.
- Quay dọc hoặc vuông đều được, nền đơn giản, đủ sáng.
- Nén cho nhẹ (dưới ~5MB mỗi file) để tải nhanh. Có thể quay muted, web sẽ tự tắt tiếng và lặp lại.

Chưa có file nào cũng không sao: trang sẽ hiện hình vẽ minh họa như bình thường,
tới khi bạn thêm clip vào thì video mới xuất hiện kèm.
